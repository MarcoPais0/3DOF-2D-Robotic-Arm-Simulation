# 3DOF Manipulator Simulation

This repository now contains the planar 3DOF 2D robotic arm project only.

## Projects

- `3DOF_2D_Robotic_Arm_Kinematic_Simulation/`: planar forward and inverse
  kinematics, workspace analysis, path tracking, and the project report

## Run

Install the project dependencies:

```bash
pip install -r 3DOF_2D_Robotic_Arm_Kinematic_Simulation/requirements.txt
```

Run the simulations from the repository root:

```bash
python 3DOF_2D_Robotic_Arm_Kinematic_Simulation/src/3DOF_2D_Simulation_Path.py
python 3DOF_2D_Robotic_Arm_Kinematic_Simulation/src/3DOF_2D_Simulation_Workspace.py
```
